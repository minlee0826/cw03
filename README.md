# Flutter Task Manager App

## Overview
This is a Flutter task management app using Firebase Firestore for real-time CRUD operations and nested subtasks.

## Features
- Add tasks
- Real-time task updates (StreamBuilder)
- Toggle task completion
- Delete tasks
- Nested subtasks (add, toggle, delete)
- Search/filter tasks
- Dark mode support

## Firebase Setup
1. Create a Firebase project
2. Run `flutterfire configure`
3. Enable Firestore in test mode
4. Run `flutter pub get`
5. Run `flutter run`

## Enhanced Features
1. Search functionality to filter tasks
2. Dark mode using system theme
